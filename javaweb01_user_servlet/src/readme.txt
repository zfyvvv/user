1.项目说明：
	1.1使用Servlet+JDBC(原生)+JSP技术，搭建简单的javaweb项目，数据库使用的Mysql;
	1.2在一个servlet中通过前端控制器思想，实现登录，增删改查等常规操作（当然也可以通过多个servlet实现！）；
	1.3	使用日志进行登录日志记录；
2.项目使用：
	2.1新建数据库t_user表，三个字段(uid,uname,pwd)；
	2.2在Tomcat下启动该项目，在浏览器输入：localhost:8080/user/login.jsp
	2.3登录成功后，可以进行CURD操作；
3项目改进：
	3.1当然该项目在异常处理，多表联合查询，数据库事务等方面，需要改进；
